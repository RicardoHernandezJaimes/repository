Pasos para instalar  La aplicacion
1.   cree en posgretsql una base de datos llamada prueba 
2.  dentro de la base de daros de prueba cree la tabla employee  con los siguiente campos id, nombre_completo, funciones 
3. Descargue los proyectos del back y del front 
4.  importe el proyecto del back  en eclipse  y ejecutelo  con spring boot
5.  abra el proyecto del front en visual studio code y   en la terminal ejecute el siguiente comando: npm install y ejecutelo  en la terminal con el comando ng serve -o
